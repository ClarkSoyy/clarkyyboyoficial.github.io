Clark Dark Mode

Dark mode is a user interface (UI) option that displays predominantly dark colors in an application or operating system. It is designed to create a more visually comfortable and less glaring experience, particularly in low-light conditions or when using devices with OLED or AMOLED screens. Traditionally, most user interfaces have featured light or bright color schemes, with white or light backgrounds and dark text. Dark mode, on the other hand, flips this color scheme by using predominantly dark backgrounds and light-colored text or elements.
